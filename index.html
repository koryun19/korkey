Инструкции по установке и подключению

Для iOS:

1. Скачайте приложение Outline из App Store.

2. Откройте приложение Outline.

3. Нажмите "Добавить сервер" (Add Server) и вставьте ключ, скопированный с этой страницы.

4. Нажмите "Подключиться" (Connect).
